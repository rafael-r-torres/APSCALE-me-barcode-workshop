Module 2: RDP Classifier (via DADA2)

Welcome to Module 2. Since APSCALE has already generated our representative sequences (ESVs or OTUs), we do not need to run the full DADA2 error-correction pipeline. Instead, we transition to R exclusively to utilize DADA2's implementation of the RDP Classifier for high-confidence taxonomic assignments.

Executing the RDP Taxonomic Assignment

This process involves fetching the appropriate marker-specific script, setting up a dedicated folder, and formatting the databases so the DADA2 package can accurately classify our sequences.
