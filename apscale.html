Module 1: APSCALE Processing

Welcome to the first stage of our pipeline. APSCALE handles the computationally intensive task of merging raw paired-end reads, quality filtering, and generating Exact Sequence Variants (ESVs) or Operational Taxonomic Units (OTUs).

Developer Mode: Click on the images to set the exact mouse click locations! The animation will update instantly and save to your browser.

1. Launching the Interface

APSCALE is a powerful Python tool that features a beautiful browser-based GUI. To make things easy, we bypass the terminal by using a .bat shortcut. Double-clicking this automatically boots up the conda environment and launches the dashboard.

2. Creating Your Project Space

Before doing any bioinformatics, APSCALE needs to build its dedicated folder tree. Through the GUI, simply point it to your main working directory and initialize the project structure.

3. Raw Data Formatting & Execution

Raw data from sequencing facilities is usually nested deeply. We use a custom Python script that iterates over all nested folders, renames them to standard _R1/_R2 formats, and copies them out. Execute this script via the terminal.

4. Data Import & Verification

Your files are now perfectly renamed and gathered in one flat directory. The final step is importing them into the APSCALE folder where the pipeline expects to find them: 02_demultiplexing/data/.

5. ESV/OTU Processing Pipeline

This core execution phase sequentially handles Paired-End Merging, Primer Trimming, Quality Filtering, Dereplication, and finally OTU Clustering (or ESV generation). Simply configure the parameters for each module in the GUI and let APSCALE process them automatically.

6. BLAST Taxonomic Assignment

With our ESVs generated, we must classify them taxonomically. This requires fetching the appropriate reference databases (e.g. MIDORI, SILVA) and executing Megablast to align and classify our sequence variants.

6.1 Generating & Updating Databases

To ensure precise taxonomic assignment, it is highly recommended to update or generate new reference databases. This can be done directly through the interface to fetch the latest data.

6.2 Running the Assignment

Once the databases are ready, we can proceed with aligning and classifying our sequences.
